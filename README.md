# php101


PHP - Hypertext Preprocessor


https://www.runoob.com/php/php-intro.html


1. php文件可包含文本、html、js、php
2. php 可直接在 server 執行，返回純 html
3. 默認副檔名 `.php`
4. 可生成動態頁面
5. 可收集表單
6. 可收發 cookie, crud 資料庫, 可輸出圖像、pdf、flash、HTML, XHTML, XML
